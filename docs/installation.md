## 系统要求

- Autodesk 3ds Max 2018+
- Windows 10/11 64位系统
- 推荐8GB+内存

### 安装步骤

#### 手动安装
方式一1.（推荐！）
直接将 AutoExportTool_Pro.mzp 文件 拖入 MAX 
，自动安装插件，并添加工具栏菜单（方便下次快捷打开插件）
方式二2. 
下载插件文件 `AutoExportTool_Pro.mzp`
,然后将文件复制到3ds Max脚本目录：
C:\Program Files\Autodesk\3ds Max XXXX\scripts
方式三3.
在3ds Max中运行脚本：
```maxscript
fileIn "AutoExportTool_Pro.mzp"
